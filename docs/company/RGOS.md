# Purpose

The Reclaim Golf Operating System (RGOS) defines how Reclaim Golf Performance LLC operates.

It serves as the company's central operating framework and establishes the standards for documentation, decision making, knowledge management, continuous improvement, and long-term strategic direction.

RGOS is the primary source of truth for the organization.

Every major decision, document, methodology, and operational process should align with this document.
